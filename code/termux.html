Termux configuration

Create the following ~/.termux/termux.properties in Termux to disable phone vibration and add more keys to the touch keyboard: 

bell-character = ignore
extra-keys = [[ \
  'TAB', 'CTRL', 'ALT', '-', '/', \
  'LEFT', 'DOWN', 'UP', 'RIGHT', \
  'DEL', 'HOME', 'END' \
]]
